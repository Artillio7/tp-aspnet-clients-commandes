# TP ASP.NET — Gestion des Clients et Commandes

Ce dépôt contient le briefing de mission initial (`initial.md`) pour développer une API REST ASP.NET Core permettant de gérer des clients et leurs commandes, avec validations (DataAnnotations + FluentValidation), EF Core et SQL Server.

## Contenu
- `initial.md`: briefing détaillé (objectifs, modélisation, EF Core, contrôleurs, erreurs, cascade, tests).
- `.gitignore`: configuration pour ignorer les fichiers de build et temporaires.

## Démarrage (prévu)
- Initialiser le projet API: `dotnet new webapi`.
- Ajouter les packages NuGet (EF Core, FluentValidation).
- Modéliser les entités, configurer `AppDbContext`, créer migrations.
- Implémenter les contrôleurs et les validations.
